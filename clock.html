Project details

This analog clock is built using HTML, CSS, and JavaScript. The clock has three needles for hours, minutes, and seconds, and is accompanied by tick marks representing each hour. In addition, there is a navigation bar at the top of the page, which includes links to various social media platforms, as well as an option to send feedback via email. The custom CSS and JS files are also linked in the head section of the HTML file.
